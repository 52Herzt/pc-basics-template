# PC 应用基础模板
## Vue3 + pinia + Antdesign 构建基础框架
```
集成 Eslint + Stylelint + Prettierrc 等规范配置实现语法格式化
集成 postcss-pxtorem + lib-flexible 实现终端适配
集成 Vue-i18n 国际化配置
支持多种主题切换及自定义配置
```
